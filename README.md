## 🛠️ Tech Stack & Key Features

### Technologies Used
*   **Structure:** Semantic HTML5 for modern SEO standards.
*   **Styling:** Custom Vanilla CSS featuring CSS Custom Properties (Variables), custom scrollbars, and grid/flexbox layouts.
*   **Interactivity:** Vanilla JavaScript for dynamic DOM manipulation and animations.
*   **Visual Effects:** HTML5 Canvas API for interactive background particle connections.

### Key Features
*   **Interactive Canvas Background:** Dynamic node particles that move and connect based on mouse proximity.
*   **Custom Hover Cursor:** Dual-layered interactive cursor with trailing circle ring scaling on interactive elements.
*   **On-Scroll Animation Reveal:** Smooth intersection-observer reveals for sections and elements as you scroll.
*   **Floating Status Badges:** Display of current specialization and institute with glassmorphic cards.
*   **Highly Responsive:** Tailored layouts for desktops, tablets, and mobile displays.

## 💻 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Duraimurugan-V/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Open `duraimurugan_portfolio.html` directly in any web browser, or serve it using a local development server like VS Code Live Server.
